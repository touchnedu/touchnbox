문제 내용

예제 문제 하나를 만들어보겠습니다.

$1/2 + 2/3 = 5/6$

라는 문제가 있고 여기에서 네모(답)가 $2/3$라고 가정하면

1/2 + 2/3 = 5/6 이렇게 작성하면 됩니다.

그러나 저희의 경우 네모가 들어가야 하기 때문에 그 부분을 분리해야 합니다.

1/2 +

2/3

=

5/6

와 같이 네모가 될 부분을 별도로 분리하고 답 부분 체크를 하면 됩니다.


그러나 $2/3$에서 $2$가 네모가 되어야 한다면(분수 내에 네모가 들어가는 경우) 별도의 코드를 작성해야 합니다.

$ { 5 - 3 } / 2 $ 의 경우를 예로 들어서 작성해보겠습니다. 이때 코드는,

{ 5 - 3 } / 2 가 됩니다. (띄어쓰기는 가독성을 위해 임의적으로 한 겁니다)

여기에서 $3$이 네모여야 한다고 가정하면

[분수 내 네모일 경우]를 체크하고 값만 입력하는 칸에는 3을,

셀렉트 박스에서 네모 번호를 선택하고 정답 입력란에 정답을 입력한 후에

'생성' 버튼을 누르면 아래와 같은 별도의 코드가 만들어집니다. (값을 3, 네모 번호를 2, 정답을 3로 임의로 입력함.)

{\html'&lt;span id="qPosition02" ans="3"&gt;3&lt;/span&gt;'}

이 전체 코드를 복사해서 3 대신 집어넣으면 됩니다.

{ 5 - 3 } / 2

↓↓↓↓↓↓

{ 5 - {\html'&lt;span id="qPosition02" ans="3"&gt;3&lt;/span&gt;'} } / 2

이 경우는 값, 네모 번호, 정답에 해당하는 코드를 직접 붙여넣었기 때문에

별도로 답체크를 하지 않습니다.

※ 정답은 앱 실행 후 마이스크립트 영역에서 답을 작성 후 뜨는 미니 팝업창의 내용을 적으세요.

   예를 들어 $11/21$ 의 경우 앱에서 [[11]/[21]]이라는 값이 반환되기 때문에

   앱과 동기화하기 위해 정답 칸에는 [[11]/[21]] 로 작성해야 합니다.
